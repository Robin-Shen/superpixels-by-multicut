GRBModel::getGenConstrAbs()

Retrieve the data of a general constraint of type ABS. Calling this function for a general constraint of different type leads to an exception. You can query the GenConstrType attribute to determine the type of the general constraint.

See also addGenConstrAbs for a description of the semantics of this general constraint type.

void getGenConstrAbs ( GRBGenConstr genc,
    GRBVar* resvarP,
    GRBVar* argvarP )

    Arguments:

    genc: The index of the general constraint.

    Any combination of the following two arguments can be NULL.

    resvarP: Pointer to store the resultant variable of the constraint.

    argvarP: Pointer to store the argument variable of the constraint.
