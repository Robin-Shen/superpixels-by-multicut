Model.getGenConstrs()

getGenConstrs ( )

Retrieve a list of all general constraints in the model.

Return value:

All general constraints in the model.

Example usage:

  gencons = model.getGenConstrs()
  for gc in gencons:
    print(model.getGenConstr(gc))
