GRBGenConstr

Gurobi general constraint object. General constraints are always associated with a particular model. You create a general constraint object by adding a general constraint to a model (using GRBModel.AddGenConstr), rather than by using a GRBGenConstr constructor.

General constraint objects have a number of attributes, which can be queried with the GRBGenConstr.Get method. The full list can be found in the Attributes section of this document.
