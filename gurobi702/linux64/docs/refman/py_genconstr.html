GenConstr

Gurobi general constraint object. General constraints are always associated with a particular model. You create a general constraint object by adding a general constraint to a model (using Model.addGenConstr), rather than by using a GenConstr constructor.

General constraint objects have a number of attributes, which can be queried with the GenConstr.getAttr method. The full list can be found in the Attributes section of this document.
