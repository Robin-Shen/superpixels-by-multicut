GRBModel.GetGenConstrOr()

Retrieve the data of a general constraint of type OR. Calling this function for a general constraint of different type leads to an exception. You can query the GenConstrType attribute to determine the type of the general constraint.

See also AddGenConstrOr for a description of the semantics of this general constraint type.

void GetGenConstrOr ( GRBGenConstr genc,
    out GRBVar resvar,
    out GRBVar[] vars )

    Arguments:

    genc: The index of the general constraint.

    resvar: Stores the resultant variable of the constraint.

    vars: Stores the array of operand variables of the constraint.
